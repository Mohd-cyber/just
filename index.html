🖥️ COMPONENTS OF A COMPUTER

CPU (Central Processing Unit) = Brain of computer. Has 3 parts:

  • ALU (Arithmetic Logic Unit) — does math and logic operations
  • CU (Control Unit) — directs all operations
  • Registers — tiny, fastest memory inside CPU

Memory Hierarchy (fastest → slowest, smallest → largest):

Registers → Cache → RAM → ROM → Hard Disk → Cloud

Key Memory types:

  • RAM — Volatile, temporary, fast
  • ROM — Non-volatile, permanent, read-only
  • Cache — Between CPU and RAM, very fast
  • Virtual Memory — Part of HDD used as RAM when RAM is full

🌐 NETWORK TOPOLOGIES — draw these!

Star ⭐ — All nodes connect to central hub/switch. Hub failure = network down. Easy to manage.

Bus 🚌 — Single backbone cable. Simple, cheap. Cable failure = all down.

Ring 🔄 — Each node connects to exactly 2 others. Data travels in one direction.

Mesh 🕸️ — Every node connects to every other. Most reliable, most expensive. Used in WANs.

Tree 🌲 — Hierarchical star topology. Root node + branches.

"Which topology requires a central controller or hub?" → STAR topology (asked in PYQ 2022-23)

"A pointer is just a variable that stores an address. Say it with me. An address. That's it. Everything else follows from that one idea."

📌 POINTER FUNDAMENTALS

CThe classic pointer program (PYQ 2022-23 Section C)
#include <stdio.h>
int main() {
    int x = 5;
    int *p = &x;       // p stores the address of x

    printf("Value of x: %d\n", x);      // 5
    printf("Value of p: %p\n", p);      // address of x (e.g., 0x7fff...)
    printf("Value at p: %d\n", *p);     // 5  (dereferencing)

    *p = 10;  // change x THROUGH the pointer
    printf("New x: %d\n", x);           // 10
    return 0;
}
PYQ Section A: int x=5; int *p = &x; printf("%p", p); → prints the memory address of x, NOT 5.

🔴 4 TYPES OF POINTERS — All 4 asked in PYQ 2024-25

POINTER TYPEDEFINITIONDANGER
NULL Pointer Points to memory address 0. Initialized as int *p = NULL; Safe — dereferencing causes crash but it's defined behavior
Wild Pointer Declared but NOT initialized. Points to random garbage address. ⚠️ Extremely dangerous — can corrupt any memory location
Dangling Pointer Pointer to memory that has been freed/deallocated. ⚠️ Very dangerous — memory is no longer valid
Void Pointer Generic pointer: void *p; — can hold address of any type Must cast before dereferencing
"NULL pointer points to the 0th memory address" → FALSE. NULL pointer points to address 0 (which is not a valid/accessible address). You cannot use it. (PYQ CA 2022-23)

📞 CALL BY VALUE vs CALL BY REFERENCE — #1 long answer topic

CSwap using Call by VALUE (does NOT work)
void swap(int a, int b) {
    int temp = a;
    a = b;
    b = temp;
    // a and b are COPIES — original unchanged!
}
int main() {
    int x=5, y=10;
    swap(x, y);
    printf("%d %d", x, y); // Still 5 10 !
}
CSwap using Call by REFERENCE (works!)
void swap(int *a, int *b) {
    int temp = *a;
    *a = *b;
    *b = temp;
    // Modifying actual memory → works!
}
int main() {
    int x=5, y=10;
    swap(&x, &y);   // pass ADDRESSES
    printf("%d %d", x, y); // 10 5 ✓
}
FEATURECALL BY VALUECALL BY REFERENCE
What is passedCopy of valueAddress of variable
Original changed?NOYES
MemoryMore (extra copy)Less
Use caseWhen original must not changeWhen you need to modify original
"An entire array is always passed by ___ to a called function." → Call by reference (address) (PYQ CA 2022-23)

🫧 BUBBLE SORT — Write this from memory

CBubble Sort (PYQ 2023-24, 2024-25)
#include <stdio.h>
int main() {
    int arr[] = {64, 25, 12, 22, 11};
    int n = 5, i, j, temp;
    
    for(i = 0; i < n-1; i++) {
        for(j = 0; j < n-i-1; j++) {
            if(arr[j] > arr[j+1]) {
                // Swap adjacent elements
                temp = arr[j];
                arr[j] = arr[j+1];
                arr[j+1] = temp;
            }
        }
    }
    // Output: 11 12 22 25 64
    return 0;
}

How to remember: Compare adjacent pairs. Largest "bubbles up" to end each pass. Outer loop = n-1 passes. Inner loop = n-i-1 comparisons.

🏆 LARGEST ELEMENT using POINTER (PYQ Section C)

    int max = *p;    // assume first is max

}

🧠 DYNAMIC MEMORY ALLOCATION — malloc, calloc, free

CDMA functions — stdlib.h required
#include <stdio.h>
#include <stdlib.h>    // REQUIRED for DMA!

int main() {
    int n = 5;
    
    // malloc: allocates, does NOT initialize (garbage)
    int *p1 = (int*) malloc(n * sizeof(int));
    
    // calloc: allocates AND initializes to 0
    int *p2 = (int*) calloc(n, sizeof(int));
    
    // realloc: resize existing block
    p1 = (int*) realloc(p1, 10 * sizeof(int));
    
    // Always free when done!
    free(p1);
    free(p2);
    return 0;
}
FUNCTIONSYNTAXINITIALIZES?ARGS
mallocmalloc(size)NO (garbage)1 arg: total bytes
calloccalloc(n, size)YES (zeros)2 args: count, size each
reallocrealloc(ptr, newsize)Preserves old data2 args: ptr, new size
freefree(ptr)Deallocates memory1 arg: pointer

📝 MUST-WRITE PROGRAMS — Master these 8

C#1: Armstrong Number Check (PYQ 2022-23)
#include <stdio.h>
int main() {
    int n, original, rem, result = 0;
    printf("Enter: "); scanf("%d", &n);
    original = n;
    while (original != 0) {
        rem = original % 10;
        result += rem * rem * rem;  // cube of each digit
        original /= 10;
    }
    if (result == n) printf("%d is Armstrong\n", n);
    else printf("%d is NOT Armstrong\n", n);
    return 0;
}
// 153 = 1³+5³+3³ = 1+125+27 = 153 ✓
C#2: Palindrome Check (PYQ 2022-23)
#include <stdio.h>
int main() {
    int n, reversed = 0, original, rem;
    printf("Enter: "); scanf("%d", &n);
    original = n;
    while (n != 0) {
        rem = n % 10;
        reversed = reversed * 10 + rem;
        n /= 10;
    }
    if (original == reversed) printf("Palindrome\n");
    else printf("Not Palindrome\n");
    return 0;
}
// 121 reversed = 121 → Palindrome
